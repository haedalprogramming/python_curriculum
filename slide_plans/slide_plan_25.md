| **11** | 실습 | **문제 분석** 📋 필요한 개념: - 문자열 뒤집기 - 대소문자 무시 - 공백/특수문자 제거 | 분석 |
| **12** | 실습 | **간단한 버전** ```python def is_palindrome_simple(s):     return s == s[::-1] print(is_palindrome_simple("기러기"))  # True print(is_palindrome_simple("파이썬"))  # False ``` 문자열 뒤집기: `[::-1]` | 간단 버전 |
| **13** | 실습 | **고급 버전** ```python def is_palindrome(s):     # 소문자로 변환, 알파벳/숫자만 추출     cleaned = ""     for char in s.lower():         if char.isalnum():  # 알파벳 또는 숫자             cleaned += char     return cleaned == cleaned[::-1] # 테스트 print(is_palindrome("A man a plan a canal Panama")) # True ``` | 고급 버전 |
| **14** | 코드 | **리스트 컴프리헨션 버전** ```python def is_palindrome(s):     cleaned = [c for c in s.lower() if c.isalnum()]     return cleaned == cleaned[::-1] ``` 한 줄로 정리! | 컴프리헨션 |
| **15** | 정리 | **사용한 개념** ✅ 슬라이싱: `[::-1]` 뒤집기 ✅ lower(): 소문자 변환 ✅ isalnum(): 알파벳/숫자 확인 ✅ 리스트 컴프리헨션 | 개념 정리 |

### PART 4. 챌린지 3 - 숫자 야구 (Challenge 3) : 8장

**목표: 종합적인 게임 프로그램**

| 번호 | 유형 | 화면 구성 및 내용 | 디자인/연출 포인트 |
|------|------|-------------------|-------------------|
| **16** | 안내 | **챌린지 3: 숫자 야구 게임** ⚾ 3자리 숫자 맞추기! - 스트라이크: 숫자와 위치 모두 일치 - 볼: 숫자만 일치 - 아웃: 모두 불일치 예: 정답 123, 추측 132 → 1S 2B | 게임 규칙 |
| **17** | 실습 | **필요한 기능** 📋 1. 랜덤 3자리 숫자 생성 (중복 없이) 2. 사용자 입력 받기 3. 스트라이크/볼 계산 4. 결과 출력 5. 정답까지 반복 | 기능 분석 |
| **18** | 실습 | **숫자 생성** ```python import random def generate_number():     digits = list(range(10))     random.shuffle(digits)     # 첫 자리는 0이 아니어야 함     if digits[0] == 0:         digits[0], digits[1] = digits[1], digits[0]     return digits[:3] secret = generate_number() print(secret)  # 예: [3, 5, 7] ``` | 코드 블록 |
| **19** | 실습 | **스트라이크/볼 계산** ```python def check_guess(secret, guess):     strike = 0     ball = 0     for i in range(3):         if guess[i] == secret[i]:             strike += 1         elif guess[i] in secret:             ball += 1     return strike, ball ``` | 코드 블록 |
| **20** | 실습 | **게임 루프** ```python def play_game():     secret = generate_number()     attempts = 0     print("⚾ 숫자 야구 게임!")     print("3자리 숫자를 맞춰보세요! (중복 없음)\\n")     while True:         guess_str = input("추측: ")         if len(guess_str) != 3 or not guess_str.isdigit():             print("3자리 숫자를 입력하세요!")             continue         guess = [int(d) for d in guess_str]         attempts += 1         s, b = check_guess(secret, guess)         if s == 3:             print(f"🎉 정답! {attempts}번 만에 맞췄습니다!")             break         elif s == 0 and b == 0:             print("아웃!")         else:             print(f"{s}스트라이크 {b}볼") play_game() ``` | 게임 루프 |
| **21** | 완료 | **실행 결과** ``` ⚾ 숫자 야구 게임! 3자리 숫자를 맞춰보세요! (중복 없음) 추측: 123 1스트라이크 0볼 추측: 145 0스트라이크 1볼 추측: 167 2스트라이크 0볼 추측: 187 🎉 정답! 4번 만에 맞췄습니다! ``` | 터미널 결과 |
