### PART 3. 리스트 메서드 (Concept 2) : 6장

**목표: 리스트 조작 메서드 학습**

| 번호 | 유형 | 화면 구성 및 내용 | 디자인/연출 포인트 |
|------|------|-------------------|-------------------|
| **11** | 개념 | **리스트 메서드 소개** 리스트에게 명령하는 방법! `리스트.메서드()` 형태로 사용 | 메서드 개념 소개 |
| **12** | 코드 | **추가: append(), insert()** ```python fruits = ["사과", "바나나"] fruits.append("오렌지")  # 맨 뒤에 추가 print(fruits)  # ["사과", "바나나", "오렌지"] fruits.insert(1, "포도")  # 1번 위치에 삽입 print(fruits)  # ["사과", "포도", "바나나", "오렌지"] ``` | 코드 + 추가 위치 표시 |
| **13** | 코드 | **삭제: remove(), pop()** ```python fruits = ["사과", "바나나", "오렌지"] fruits.remove("바나나")  # 값으로 삭제 print(fruits)  # ["사과", "오렌지"] last = fruits.pop()  # 맨 뒤 빼기 print(last)    # 오렌지 print(fruits)  # ["사과"] ``` | 코드 + 삭제 화살표 |
| **14** | 코드 | **정렬: sort(), reverse()** ```python nums = [3, 1, 4, 1, 5] nums.sort()     # 오름차순 정렬 print(nums)     # [1, 1, 3, 4, 5] nums.reverse()  # 순서 뒤집기 print(nums)     # [5, 4, 3, 1, 1] ``` | 코드 + 결과 |
| **15** | 코드 | **기타 유용한 메서드** ```python nums = [1, 2, 3, 2, 1] print(nums.count(2))  # 2 (2가 몇 개?) print(nums.index(3))  # 2 (3의 위치) nums.clear()           # 모두 삭제 print(nums)            # [] ``` | 코드 + 결과 |
| **16** | 정리 | **리스트 메서드 정리** | 추가 | `append()`, `insert()` | | 삭제 | `remove()`, `pop()`, `clear()` | | 정렬 | `sort()`, `reverse()` | | 기타 | `count()`, `index()` | | 메서드 정리표 |
