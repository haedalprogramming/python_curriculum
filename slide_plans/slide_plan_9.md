|------|------|-------------------|-------------------|
| **05** | 개념 | **함수(Function)란?** 📝 요리 레시피처럼! 1. 재료를 넣으면 (입력) 2. 레시피대로 요리하고 (처리) 3. 음식이 나온다 (출력) **코드를 묶어서 이름 붙인 것!** | 레시피 비유 그림 |
| **06** | 개념 | **이미 쓰고 있는 함수들!** ```python print("안녕")     # 출력 함수 input("이름: ")   # 입력 함수 len("Hello")      # 길이 함수 int("123")        # 변환 함수 ``` 이것들 모두 함수예요! | 내장 함수 예시 |
| **07** | 코드 | **함수 만들기 (정의)** ```python def 함수이름():     실행할 코드 # 예시 def say_hello():     print("안녕하세요!") ``` `def` = define(정의하다) | 문법 구조 |

| **09** | 주의 | **⚠️ 정의 vs 호출** ```python # 정의만 하면 실행 안 됨! def greet():     print("Hi!") # 호출해야 실행됨! greet() ``` 정의 = 레시피 작성 호출 = 실제 요리 | 정의/호출 비교 |
| **10** | 실습 | **여러 줄 함수** ```python def introduce():     print("=" * 20)     print("안녕하세요!")     print("저는 파이썬입니다.")     print("=" * 20) introduce() ``` | 코드 + 결과 |

### PART 3. 매개변수 (Concept 2) : 6장

**목표: 함수에 값 전달하기**

| 번호 | 유형 | 화면 구성 및 내용 | 디자인/연출 포인트 |
|------|------|-------------------|-------------------|
| **11** | 개념 | **매개변수란?** 📦 함수에 전달하는 재료! ```python def greet(name):  # name = 매개변수     print(f"안녕, {name}!") greet("철수")  # 안녕, 철수! greet("영희")  # 안녕, 영희! ``` 같은 함수, 다른 결과! | 재료 전달 비유 |
| **12** | 코드 | **매개변수 여러 개** ```python def add(a, b):     print(f"{a} + {b} = {a + b}") add(3, 5)   # 3 + 5 = 8 add(10, 20) # 10 + 20 = 30 ``` 쉼표로 구분! | 코드 + 결과 |
| **13** | 코드 | **기본값 설정** ```python def greet(name, msg="안녕하세요"):     print(f"{name}님, {msg}!") greet("철수")           # 철수님, 안녕하세요! greet("영희", "반가워요") # 영희님, 반가워요! ``` 기본값 있으면 생략 가능! | 코드 + 기본값 강조 |
| **14** | 코드 | **키워드 인수** ```python def info(name, age, city):     print(f"{name}, {age}살, {city}") # 키워드로 순서 상관없이! info(age=20, city="서울", name="민수") ``` | 코드 + 순서 무관 |
| **15** | 주의 | **⚠️ 매개변수 vs 인수** ```python def greet(name):  # 매개변수 (parameter)     print(name) greet("철수")     # 인수 (argument) ``` 정의할 때 = 매개변수 호출할 때 = 인수 | 용어 구분 |
| **16** | 실습 | **실습: 사칙연산 함수** ```python def calc(a, b, op):     if op == "+":         print(a + b)     elif op == "-":         print(a - b)     elif op == "*":         print(a * b)     elif op == "/":         print(a / b) calc(10, 3, "+")  # 13 calc(10, 3, "*")  # 30 ``` | 코드 블록 |

### PART 4. 반환값 (Concept 3) : 5장

**목표: 함수에서 값 돌려받기**

| 번호 | 유형 | 화면 구성 및 내용 | 디자인/연출 포인트 |
|------|------|-------------------|-------------------|
| **17** | 개념 | **return이란?** 🎁 함수가 결과를 돌려주는 것! ```python def add(a, b):     return a + b  # 결과 반환! result = add(3, 5) print(result)  # 8 ``` `return` = 결과를 돌려줘! | 선물 비유 |
| **18** | 코드 | **return 활용** ```python def square(n):     return n ** 2 x = square(4)  # 16 y = square(5)  # 25 print(x + y)   # 41 ``` 반환된 값을 변수에 저장! | 코드 + 결과 |
| **19** | 코드 | **여러 값 반환** ```python def calc(a, b):     return a + b, a - b, a * b, a / b add, sub, mul, div = calc(10, 3) print(add, sub, mul, div) # 13 7 30 3.333... ``` 쉼표로 여러 값 반환! | 코드 + 튜플 언패킹 |
| **20** | 주의 | **print vs return** ```python # print: 화면에 출력만 def add1(a, b):     print(a + b) # return: 값을 돌려줌 def add2(a, b):     return a + b result1 = add1(1, 2)  # None result2 = add2(1, 2)  # 3 ``` | print/return 비교 |
| **21** | 코드 | **return 후 코드** ```python def test():     print("A")     return "끝"     print("B")  # 실행 안 됨! print(test())  # A, 끝 ``` return 하면 함수 종료! | 코드 + 주의 |
