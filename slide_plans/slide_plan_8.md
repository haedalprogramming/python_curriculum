### PART 2. while 기초 (Concept 1) : 6장

**목표: while문의 기본 구조 이해**

| 번호 | 유형 | 화면 구성 및 내용 | 디자인/연출 포인트 |
|------|------|-------------------|-------------------|
| **05** | 개념 | **for vs while** | | for | while | |---|---| | 정해진 횟수 반복 | 조건이 참인 동안 반복 | | "10번 반복해" | "맞출 때까지 반복해" | while = 조건 기반 반복! | 비교표 |
| **06** | 개념 | **while문 구조** ```python while 조건:     반복할 코드 ``` 조건이 **True인 동안** 반복! False가 되면 탈출! | 구조 도식 |
| **07** | 코드 | **while문 기본 예제** ```python count = 0 while count < 5:     print(count)     count += 1  # 중요! # 0, 1, 2, 3, 4 ``` count가 5 미만인 동안 반복! | 코드 + 흐름도 |
| **08** | 주의 | **⚠️ 무한 루프 주의!** ```python count = 0 while count < 5:     print(count)     # count += 1 빠짐! ``` 조건이 영원히 True면 멈추지 않아요! Ctrl+C로 강제 종료 | 경고 박스 |
| **09** | 코드 | **카운트다운** ```python n = 5 while n > 0:     print(n)     n -= 1 print("발사! 🚀") # 5, 4, 3, 2, 1, 발사! 🚀 ``` | 코드 + 카운트다운 |
| **10** | 비교 | **for로 바꾸면?** ```python # while 버전 n = 5 while n > 0:     print(n)     n -= 1 # for 버전 for n in range(5, 0, -1):     print(n) ``` 같은 결과, 다른 방법! | 코드 비교 |

### PART 3. while True 패턴 (Concept 2) : 6장

**목표: 무한 루프와 break 활용**

| 번호 | 유형 | 화면 구성 및 내용 | 디자인/연출 포인트 |
|------|------|-------------------|-------------------|
| **11** | 개념 | **while True란?** ```python while True:     # 무한 반복!     if 탈출조건:         break  # 탈출! ``` 일단 무한 반복 → 조건 맞으면 break로 탈출! | 무한 루프 도식 |
| **12** | 코드 | **비밀번호 확인** ```python password = "1234" while True:     pw = input("비밀번호: ")     if pw == password:         print("로그인 성공!")         break     print("틀렸습니다. 다시 시도하세요.") ``` | 코드 + 흐름 |
| **13** | 코드 | **메뉴 선택** ```python while True:     print("1. 게임 시작")     print("2. 설정")     print("3. 종료")     choice = input("선택: ")     if choice == "3":         print("안녕히 가세요!")         break     elif choice == "1":         print("게임 시작!")     elif choice == "2":         print("설정 화면") ``` | 코드 블록 |
| **14** | 개념 | **왜 while True를 쓸까?** ✅ 반복 횟수를 모를 때 ✅ 사용자 입력을 계속 받을 때 ✅ 특정 조건이 만족될 때까지 게임, 메뉴, 로그인 등에 많이 사용! | 사용 예시 |
| **15** | 코드 | **숫자 수집하기** ```python numbers = [] while True:     num = input("숫자 입력 (q=종료): ")     if num == 'q':         break     numbers.append(int(num)) print(f"입력한 숫자: {numbers}") print(f"합계: {sum(numbers)}") ``` | 코드 + 결과 |
| **16** | 실습 | **연습: 양수만 받기** ```python while True:     n = int(input("양수를 입력: "))     if n > 0:         print(f"{n} 입력 완료!")         break     print("양수가 아닙니다. 다시!") ``` | 실습 코드 |
