### PART 2. input() 함수 (Concept 1) : 6장

**목표: 사용자 입력 받기**

| 번호 | 유형 | 화면 구성 및 내용 | 디자인/연출 포인트 |
|------|------|-------------------|-------------------|
| **05** | 개념 | **input()이란?** 🎤 사용자의 입력을 기다리는 함수 `input("메시지")` 메시지 보여주고 → 입력 기다리기 | 마이크 비유 |
| **06** | 코드 | **input() 기본 사용** ```python name = input("이름이 뭐예요? ") print(f"안녕, {name}!") ``` 입력한 값이 `name`에 저장! | 코드 + 결과 |
| **07** | 주의 | **⚠️ input()은 항상 문자열!** ```python age = input("나이: ")  # "15" print(type(age))  # <class 'str'> ``` 숫자를 입력해도 문자열로 저장! | 타입 강조 |
| **08** | 개념 | **형 변환 (Type Conversion)** ```python # 문자열 → 정수 age = int(input("나이: ")) # 문자열 → 실수 height = float(input("키: ")) ``` `int()`, `float()`로 숫자 변환! | 변환 화살표 |
| **09** | 코드 | **형 변환 실습** ```python num1 = int(input("첫 번째 숫자: ")) num2 = int(input("두 번째 숫자: ")) print(f"합: {num1 + num2}") ``` 변환 안 하면 문자열 더하기 됨! | 코드 + 올바른 결과 |
| **10** | 주의 | **⚠️ ValueError 주의!** ```python age = int(input("나이: ")) # "열다섯" 입력 → 에러! ``` 숫자가 아닌 값은 변환 불가! | 에러 메시지 |

### PART 3. print() 고급 (Concept 2) : 6장

**목표: print() 다양한 활용법**

| 번호 | 유형 | 화면 구성 및 내용 | 디자인/연출 포인트 |
|------|------|-------------------|-------------------|
| **11** | 개념 | **print() 여러 값 출력** ```python name = "철수" age = 15 print(name, age)  # 철수 15 print(name, age, sep=", ")  # 철수, 15 ``` `sep` = 구분자 (기본은 공백) | 코드 + 결과 |
| **12** | 코드 | **줄바꿈 제어: end** ```python print("Hello", end=" ") print("World") # Hello World (같은 줄에 출력) ``` 기본 `end="\n"` (줄바꿈) | 코드 + 결과 비교 |
| **13** | 코드 | **이스케이프 문자** ```python print("안녕\n하세요")  # 줄바꿈 print("탭\t간격")      # 탭 print("따옴표: \"\"")  # 따옴표 ``` `\n` 줄바꿈, `\t` 탭 | 이스케이프 문자표 |
| **14** | 코드 | **f-string 복습** ```python name = "영희" score = 95 print(f"{name}의 점수: {score}점") print(f"평균: {score/100*100:.1f}%") ``` `:.1f` = 소수점 1자리 | 코드 + 포맷팅 |
| **15** | 코드 | **정렬하기** ```python print(f"{'왼쪽':<10}|")   # 왼쪽 정렬 print(f"{'가운데':^10}|")  # 가운데 정렬 print(f"{'오른쪽':>10}|")  # 오른쪽 정렬 ``` `<` 왼쪽, `^` 가운데, `>` 오른쪽 | 코드 + 정렬 결과 |
| **16** | 코드 | **숫자 포맷팅** ```python price = 1234567 print(f"{price:,}원")  # 1,234,567원 pi = 3.141592 print(f"{pi:.2f}")  # 3.14 ``` 천 단위 쉼표, 소수점 자릿수 | 코드 + 결과 |
