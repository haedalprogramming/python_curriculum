## 3. 제작 시 유의사항 (Checklist)

1. **설치 화면 스크린샷**: 최신 버전 기준으로 캡처 (Python 3.12+, VS Code 최신)
2. **PATH 설정 강조**: "Add Python to PATH" 체크 안 하면 실행 안 됨 - 빨간 박스로 강조
3. **코드 블록**: 다크 테마(#1E1E1E) 배경, 줄 번호 표시, 구문 강조
4. **OS 구분**: Windows 기준 설명 (Mac/Linux는 별도 안내 또는 부록)
5. **용어 설명**: 터미널, 에디터, 확장 등 처음 나오는 용어는 짧게 설명
