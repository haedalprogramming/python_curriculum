### PART 4. 실습 A - 기본 도형 (Practice A) : 6장

**목표: 삼각형, 사각형, 원 그리기**

| 번호 | 유형 | 화면 구성 및 내용 | 디자인/연출 포인트 |
|------|------|-------------------|-------------------|
| **16** | 안내 | **실습 A: 기본 도형 그리기** 🔷 삼각형, 사각형, 원을 그려봐요! 반복문 활용! | 도형 이미지 |
| **17** | 실습 | **삼각형 그리기** ```python import turtle t = turtle.Turtle() t.speed(3) for _ in range(3):     t.fd(100)     t.lt(120)  # 360 / 3 = 120 turtle.done() ``` 정삼각형: 외각 120도! | 코드 + 삼각형 |
| **18** | 실습 | **사각형 그리기** ```python for _ in range(4):     t.fd(100)     t.lt(90)  # 360 / 4 = 90 ``` 정사각형: 외각 90도! | 코드 + 사각형 |
| **19** | 개념 | **정다각형 공식** 외각 = 360 / 변의 개수 - 삼각형: 360/3 = 120도 - 사각형: 360/4 = 90도 - 오각형: 360/5 = 72도 - 육각형: 360/6 = 60도 | 공식 정리 |
| **20** | 실습 | **원 그리기** ```python t.circle(50)    # 반지름 50인 원 t.circle(100)   # 반지름 100인 원 t.circle(50, 180)  # 반원 (180도) t.circle(50, 90)   # 사분원 (90도) ``` | 코드 + 원 |
| **21** | 실습 | **여러 도형 함께 그리기** ```python def draw_shape(sides, size):     angle = 360 / sides     for _ in range(sides):         t.fd(size)         t.lt(angle) draw_shape(3, 80)  # 삼각형 t.pu(); t.fd(150); t.pd() draw_shape(4, 80)  # 사각형 t.pu(); t.fd(150); t.pd() draw_shape(6, 50)  # 육각형 ``` | 함수 활용 |

| **24** | 실습 | **오각별 그리기** ```python import turtle t = turtle.Turtle() t.speed(3) t.pencolor("gold") t.fillcolor("yellow") t.begin_fill() for _ in range(5):     t.fd(100)     t.rt(144)  # 144도 회전 t.end_fill() turtle.done() ``` | 코드 + 별 |
| **25** | 실습 | **여러 개 별** ```python import random def draw_star(size):     t.begin_fill()     for _ in range(5):         t.fd(size)         t.rt(144)     t.end_fill() for _ in range(5):     x = random.randint(-200, 200)     y = random.randint(-200, 200)     t.pu()     t.goto(x, y)     t.pd()     draw_star(30) ``` | 코드 + 여러 별 |
| **26** | 실습 | **다양한 모양 별** ```python # 6각별 for _ in range(6):     t.fd(100)     t.rt(120)  # 360/6*2 = 120 # 8각별 for _ in range(8):     t.fd(80)     t.rt(135)  # 360/8*3 = 135 ``` | 다양한 별 |
