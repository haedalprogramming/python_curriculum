### PART 3. range() 함수 (Concept 2) : 6장

**목표: range()로 숫자 범위 만들기**

| 번호 | 유형 | 화면 구성 및 내용 | 디자인/연출 포인트 |
|------|------|-------------------|-------------------|
| **11** | 개념 | **range()란?** 🔢 숫자 범위를 만들어주는 함수! `range(5)` = 0, 1, 2, 3, 4 0부터 시작, 5개 숫자! | 숫자 나열 |
| **12** | 코드 | **range() 기본** ```python for i in range(5):     print(i) # 0 # 1 # 2 # 3 # 4 ``` `range(n)` = 0부터 n-1까지 | 코드 + 결과 |
| **13** | 코드 | **range(시작, 끝)** ```python for i in range(1, 6):     print(i) # 1, 2, 3, 4, 5 ``` `range(a, b)` = a부터 b-1까지 | 코드 + 범위 표시 |
| **14** | 코드 | **range(시작, 끝, 간격)** ```python for i in range(0, 10, 2):     print(i)  # 0, 2, 4, 6, 8 for i in range(5, 0, -1):     print(i)  # 5, 4, 3, 2, 1 ``` 간격(step) 지정 가능! | 코드 + 결과 |
| **15** | 정리 | **range() 정리** | 형태 | 의미 | 예시 | |------|------|------| | `range(n)` | 0부터 n-1 | range(5) → 0,1,2,3,4 | | `range(a,b)` | a부터 b-1 | range(1,5) → 1,2,3,4 | | `range(a,b,c)` | a부터 b-1, c간격 | range(0,10,2) → 0,2,4,6,8 | | range 정리표 |
| **16** | 실습 | **응용: n번 반복** ```python for _ in range(3):     print("안녕하세요!") # 안녕하세요! # 안녕하세요! # 안녕하세요! ``` `_` = 변수를 안 쓸 때 | 코드 + 설명 |

### PART 6. 추가 개념 (Advanced) : 4장

| 번호 | 유형 | 화면 구성 및 내용 | 디자인/연출 포인트 |
|------|------|-------------------|-------------------|

| **30** | 실습 | **숫자 찾기 게임** ```python target = 7 for i in range(1, 11):     if i == target:         print(f"{i} 찾았다!")         break     print(f"{i}...") ``` | 코드 + 결과 |
