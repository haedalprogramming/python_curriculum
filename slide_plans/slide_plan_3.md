|------|------|-------------------|-------------------|
| **16** | 개념 | **슬라이싱이란?** 🍕 피자 자르듯 문자열을 자르기! `word[시작:끝]` 시작부터 끝-1까지 추출 | 피자 슬라이스 비유 |
| **17** | 코드 | **슬라이싱 기본** ```python word = "Python" print(word[0:2])  # Py print(word[2:5])  # tho print(word[0:6])  # Python ``` 끝 번호는 포함 안 됨! | 코드 + 범위 표시 |
| **18** | 코드 | **슬라이싱 생략** ```python word = "Python" print(word[:3])   # Pyt (처음부터) print(word[3:])   # hon (끝까지) print(word[:])    # Python (전체) ``` | 코드 + 생략 규칙 |
| **19** | 코드 | **스텝(간격) 지정** ```python word = "Python" print(word[0:6:2])  # Pto (2칸씩) print(word[::2])    # Pto print(word[::-1])   # nohtyP (역순!) ``` `[::-1]` = 뒤집기! | 코드 + 스텝 설명 |
| **20** | 실습 | **슬라이싱 연습** "Hello, World!" 에서... `[:5]` → Hello `[7:12]` → World `[::-1]` → !dlroW ,olleH | 연습 문제 |

## 3. 제작 시 유의사항 (Checklist)

1. **목걸이 비유**: 문자열 = 글자 목걸이로 일관되게 표현
2. **인덱스 0부터**: 컴퓨터는 0부터 센다는 점 반복 강조
3. **슬라이싱 끝 제외**: `[0:3]`은 0,1,2만 포함 (3 제외)
4. **f-string 권장**: 여러 포맷팅 중 f-string을 주로 사용
5. **실습 코드**: `code/3-*.py` 파일과 일치 확인
