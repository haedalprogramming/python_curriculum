## 3. 표준 슬라이드 구조 (차시당)

```
PART 1. 도입 (Intro) : 4장
├── 표지
├── 질문 (흥미 유발)
├── 미리보기 (오늘의 완성작)
└── 오늘의 미션

PART 2. 개념 학습 (Concept) : 8~12장
├── 핵심 개념 설명
├── 코드 예시 (다크 테마)
├── 비유 활용 설명
└── 개념 정리

PART 3. 실습 A (Practice A) : 6~8장
├── 실습 안내
├── 단계별 코딩
└── 실행 결과 확인

PART 4. 실습 B (Practice B) : 6~8장
├── 심화 실습 안내
├── 단계별 코딩
└── 실행 결과 확인

PART 5. 도전 과제 (Challenge) : 4~6장
├── 문제 제시
├── 힌트
├── 정답 공개
└── 해설

PART 6. 마무리 (Finish) : 4~5장
├── 오늘 배운 내용 정리
├── SOS (자주 묻는 질문)
├── 다음 차시 예고
└── 엔딩
```

### 5.1 비유 사전

| 개념 | 비유 | 이모지 |
|------|------|--------|
| 변수 | 이름표가 붙은 상자 | 📦 |
| 함수 | 요리 레시피 | 📝 |
| 반복문 | 도돌이표 / 무한 재생 | 🔁 |
| 조건문 | 갈림길 / 선택의 순간 | 🔀 |

| 문자열 | 글자 목걸이 | 📿 |

| 에러 | 선생님의 빨간펜 | 🖊️ |

- 코드 실행: "딱!", "뿅!", "짠!"

- 성공: "우와!", "와앙!"
- 오류: "어라?", "앗!"
