# 📅 Part 2. 파이썬 무기 창고 털기 (라이브러리 활용)

## 11차시. 운에 맡겨라 (`random` 모듈)
**📝 설명**
- 컴퓨터는 원래 시키는 대로만 하는 기계라서 '랜덤(무작위)'이라는 개념이 없습니다.
- 그래서 파이썬은 `random`이라는 도구를 통해 주사위를 굴리거나 제비뽑기를 하는 것처럼 무작위 숫자를 만들어냅니다.

### 💻 실습 1: 점심 메뉴 추천기 & 주사위
```python
import random

# 1. 점심 메뉴 추천
menu_list = ["짜장면", "짬뽕", "돈까스", "김치찌개", "급식"]
today_lunch = random.choice(menu_list) # 리스트 중 하나를 랜덤 선택
print(f"오늘 점심은 {today_lunch} 어때요?")

# 2. 주사위 굴리기
dice = random.randint(1, 6) # 1부터 6 사이의 정수 중 하나 랜덤
print(f"주사위 눈: {dice}")
```
### 💻 실습 2: 로또 번호 생성기
```python
import random

# 1부터 45까지의 숫자 중, 중복 없이 6개를 뽑는다.
lotto = random.sample(range(1, 46), 6)
lotto.sort() # 보기 좋게 정렬 (오름차순)

print("이번 주 로또 예상 번호:", lotto)
```

#### 20차시. 차곡차곡 쌓는 `스택 (Stack)`
- **내용**: 프링글스 통과 웹 브라우저 '뒤로 가기' 원리 (LIFO)
- **실습**: 파이썬 리스트로 스택 구현하고 `push`, `pop` 해보기

#### 21차시. 줄 서기 `큐 (Queue)`
- **내용**: 맛집 대기 줄과 프린터 인쇄 대기열 원리 (FIFO)
- **실습**: `collections` 모듈의 `deque`를 이용해 대기열 시스템 만들기

#### 22차시. 보물 찾기 1: `선형 탐색 (Linear Search)`

- **실습**: 리스트에서 특정 값의 위치(인덱스) 찾기, 최댓값/최솟값 찾기

#### 23차시. 보물 찾기 2: `이진 탐색 (Binary Search)`
- **내용**: 업/다운 게임의 원리, 반씩 쪼개서 찾기 (정렬의 중요성)
- **실습**: 1000개의 숫자 중 특정 숫자를 10번 안에 찾아내는 원리 구현

#### 24차시. 줄 세우기: `정렬 (Sorting)` 맛보기
- **내용**: 데이터를 순서대로 나열하는 법, 버블 정렬의 원리 (이웃과 비교하기)
- **실습**: 우리 반 키 순서대로 줄 세우기, 파이썬 내장 `sort()` 함수의 위력

#### 25차시. [종합] 알고리즘 챌린지 & 수료
