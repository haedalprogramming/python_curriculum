### PART 2. 비교 연산자 (Concept 1) : 5장

**목표: 조건을 만드는 비교 연산자 이해**

| 번호 | 유형 | 화면 구성 및 내용 | 디자인/연출 포인트 |
|------|------|-------------------|-------------------|
| **05** | 개념 | **비교 연산자란?** 🤔 두 값을 비교해서 **True(참)** 또는 **False(거짓)**을 알려줘요 마치 "맞아? 틀려?" 물어보는 것! | True/False 카드, 질문 비유 |
| **06** | 개념 | **비교 연산자 6가지** `==` 같다 (등호 2개!) `!=` 다르다 `>` 크다 `<` 작다 `>=` 크거나 같다 `<=` 작거나 같다 | 6개 카드, = vs == 강조 |
| **07** | 코드 | **비교 연산자 실습** ```python a = 10 b = 5 print(a == b)   # False print(a != b)   # True print(a > b)    # True print(a >= 10)  # True ``` | 코드 + 결과 |
| **08** | 주의 | **⚠️ 주의! = vs ==** `=` 저장하기 (대입) `==` 비교하기 (같은지 확인) ```python x = 5     # x에 5를 저장 x == 5    # x가 5와 같은지 비교 → True ``` | 빨간 박스 강조, 비교표 |
| **09** | 개념 | **문자열도 비교 가능!** ```python name = "철수" print(name == "철수")  # True print(name == "영희")  # False ``` | 코드 + 결과 |

### PART 3. if 조건문 (Concept 2) : 8장

**목표: if, elif, else 구조 이해**

| 번호 | 유형 | 화면 구성 및 내용 | 디자인/연출 포인트 |
|------|------|-------------------|-------------------|
| **10** | 개념 | **if문이란?** 🔀 "만약 ~라면, ~해라" 갈림길에서 방향을 정하는 것! ```python if 조건:     실행할 코드 ``` | 갈림길 그림, 화살표 |
| **11** | 코드 | **if문 기본 예제** ```python age = 15 if age >= 13:     print("청소년입니다!") ``` 조건이 True면 들여쓰기된 코드 실행! | 코드 + 들여쓰기 강조 |
| **12** | 주의 | **⚠️ 들여쓰기가 중요해요!** ```python if age >= 13:     print("OK")  # 들여쓰기 4칸! ``` - 탭(Tab) 또는 스페이스 4칸 - 들여쓰기 없으면 에러! | 들여쓰기 시각화, 에러 예시 |
| **13** | 개념 | **if-else: 둘 중 하나** ```python if 조건:     # 조건이 참이면 else:     # 조건이 거짓이면 ``` "만약 ~라면... 아니면..." | 두 갈래 화살표 |
| **14** | 코드 | **if-else 예제** ```python score = 75 if score >= 60:     print("합격!") else:     print("불합격...") ``` 60점 이상이면 합격, 아니면 불합격 | 코드 + 결과 |
| **15** | 개념 | **elif: 여러 조건 검사** ```python if 조건1:     # 조건1이 참 elif 조건2:     # 조건2가 참 elif 조건3:     # 조건3이 참 else:     # 모두 거짓 ``` | 여러 갈래 화살표 |
| **16** | 코드 | **elif 예제: 학점 판별** ```python score = 85 if score >= 90:     print("A") elif score >= 80:     print("B") elif score >= 70:     print("C") else:     print("F") ``` | 코드 + 결과 "B" |
| **17** | 주의 | **조건 검사 순서가 중요!** 위에서 아래로 순서대로 검사 먼저 True가 되면 나머지는 검사 안 함! ```python # 90 이상이면 A 먼저 확인 # A가 아니면 80 이상인지 확인... ``` | 순서도 화살표 |

### PART 4. 논리 연산자 (Concept 3) : 4장

**목표: and, or, not 이해**

| 번호 | 유형 | 화면 구성 및 내용 | 디자인/연출 포인트 |
|------|------|-------------------|-------------------|
| **18** | 개념 | **논리 연산자** 조건을 합치거나 뒤집을 때 사용! `and` 그리고 (둘 다 참이면 참) `or` 또는 (하나만 참이면 참) `not` 부정 (참↔거짓 뒤집기) | 3개 카드 |
| **19** | 코드 | **and 예제** ```python age = 20 has_id = True if age >= 19 and has_id:     print("입장 가능!") ``` 19세 이상 **그리고** 신분증이 있어야 입장 | 코드 + 벤다이어그램 |
| **20** | 코드 | **or 예제** ```python day = "토요일" if day == "토요일" or day == "일요일":     print("주말이다!") ``` 토요일 **또는** 일요일이면 주말 | 코드 + 벤다이어그램 |
| **21** | 코드 | **not 예제** ```python is_raining = False if not is_raining:     print("산책 가자!") ``` 비가 오지 **않으면** 산책 | 코드 + 뒤집기 화살표 |
