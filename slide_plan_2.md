| 번호 | 유형 | 화면 구성 및 내용 | 디자인/연출 포인트 |
|------|------|-------------------|-------------------|
| **23** | 안내 | **실습 B: BMI 측정기** 📊 키와 몸무게로 BMI를 계산해요! BMI = 몸무게(kg) ÷ 키(m)² | BMI 공식 강조, 건강 이미지 |
| **24** | 개념 | **BMI 공식 파헤치기** ```python bmi = weight / (height ** 2) ``` - weight: 몸무게 (kg) - height: 키 (m 단위!) - `**2`: 제곱 | 공식 분해 설명 |
| **25** | 실습 | **1단계: 입력받기** ```python weight = float(input("몸무게(kg): ")) height = float(input("키(m): ")) ``` `float()` = 실수로 바꾸기 (소수점 가능) | 코드 + float 설명 |
| **26** | 실습 | **2단계: BMI 계산** ```python bmi = weight / (height ** 2) ``` 괄호 조심! 키를 먼저 제곱해야 해요 | 코드, 괄호 강조 |
| **27** | 실습 | **3단계: 결과 출력** ```python print("당신의 BMI:", bmi) print("반올림:", round(bmi, 1)) ``` `round(값, 자릿수)` = 반올림 함수 | 코드 + round 설명 |
| **28** | 완료 | **실행 결과** ``` 몸무게(kg): 65 키(m): 1.75 당신의 BMI: 21.22448... 반올림: 21.2 ``` 🏃 건강한 BMI! | 터미널 결과, BMI 기준표 참고 |
